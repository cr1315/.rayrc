### config

- `~/.config/niri/config.kdl`

```kdl
    keyboard {
        repeat-delay 200
        repeat-rate 50

        xkb {
            layout "jp"
        }

        // Enable numlock on startup, omitting this setting disables it.
        numlock
    }
```
